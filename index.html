RS3 Clue Scanner

Paste a screenshot, upload one, or type a clue. Text clues use OCR. Map clues use image matching against RS Wiki map clue images.

Paste screenshot here

Click this box, then press Ctrl+V. You can also drag and drop a screenshot.

Auto tries OCR first. If there is hardly any readable text, it switches to map image matching.

Pick the scroll tier before matching a map. This filters the image pool where the Wiki provides tier data, and your saved corrections also remember the selected tier.

Uses the public RuneScape Wiki MediaWiki API and links users to original RS Wiki pages. Wiki content is not copied into this app.
