# RS3 Clue Wiki Scanner PWA

This version can be installed on a phone as a PWA.

## Easiest setup

1. Upload the whole `rs3-clue-wiki-app` folder to GitHub.
2. Turn on GitHub Pages for the repository.
3. Open the GitHub Pages link on your phone.
4. Android: Chrome > three dots > Add to Home screen.
5. iPhone: Safari > Share > Add to Home Screen.

## What is included

- `index.html` - the app page
- `app.js` - clue scanning and matching logic
- `style.css` - styling
- `manifest.json` - tells the phone this is an installable app
- `service-worker.js` - makes the app installable and caches local files
- `icon-192.png` and `icon-512.png` - app icons

## Mobile camera

Use the camera file picker in the paste/upload box. It opens the phone camera where supported.
